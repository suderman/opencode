# claude-browser OpenCode Skill

This skill teaches an OpenCode agent how to ask questions through the user's logged-in Claude web session using chrome-devtools MCP.

Install:

```bash
mkdir -p ~/.config/opencode/skills/claude-browser
cp SKILL.md ~/.config/opencode/skills/claude-browser/SKILL.md
```

It enters through this Claude project URL:

```
https://claude.ai/project/019dd048-be96-755c-8349-f76d8c66ed29
```
